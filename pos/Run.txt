Instructins :
=============
1. Install Node & Yarn
2. Run ./mvnw
3. Ctrl+c to Stop run of spring boot started by mvnw
3. yarn start
4. yarn stop
5. Run again ./mvnw
6. Run ./gen-data.sh to generate bill data
7. Run ./show-bill.sh to view the generated bill
8. UI available on http://localhost:8080/
9. user name : admin  passwod : admin
